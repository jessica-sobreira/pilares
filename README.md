# Resumo - Exercícios de POO em TypeScript

## Atividade 1: Ingressos
- Criação de uma classe abstrata `Ingresso` com valor em reais e método `imprimeValor()`.
- Subclasses `Normal`, `VIP`, e `Camarote` herdam de `Ingresso` com implementações específicas.
- Demonstração de uso com instâncias das classes.

## Atividade 2: Imóveis
- Classe `Imovel` com endereço e preço.
- Subclasses `Novo` e `Velho` herdam de `Imovel` com adicional/desconto no preço e métodos específicos.
- Exemplos de instâncias e uso dos métodos.

## Atividade 3: Hierarquia de Animais
- Identificação de atributos/comportamentos comuns entre `Cachorro`, `Cavalo`, e `Gato`.
- Criação da superclasse `Animal` com métodos `comer` e `dormir`.
- Adição de comportamentos específicos para cada animal (`latir`, `galopar`, `miar`).
- Demonstração de uso com instâncias das classes.
